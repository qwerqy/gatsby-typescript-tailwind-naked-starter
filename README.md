# gatsby-ts-naked-starter

Included dependencies
- Typescript 
- Tailwindcss
- Jest
- react-helmet
- gatsby plugins

## Use this starter

`$ gatsby new project-name https://github.com/qwerqy/gatsby-typescript-tailwind-naked-starter.git`

## Installation
This project uses `yarn`.

`$ yarn install` 

## Development
`$ yarn develop`
